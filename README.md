# 🎬 Movie Info Extractor - AI Powered Movie Information Assistant

An AI-powered movie information extraction application that uses Large Language Models to extract and organize useful information about movies from user queries.

Instead of manually searching through multiple sources, users can simply ask questions related to movies and get structured, meaningful responses generated using Generative AI techniques.

This project was built to explore how LLMs can be used for information extraction, prompt engineering, and building intelligent AI applications.

---

## ✨ Features

- 🎥 Extracts useful movie-related information using AI

- 🧠 LLM-powered responses for understanding natural language queries

- 📝 Provides structured and meaningful information instead of plain text responses

- ⚡ Fast AI-generated answers

- 🎨 Simple and interactive user experience

---

## 🚀 How It Works

1. User enters a movie-related query.

2. The application processes the input and sends it to the Large Language Model.

3. Custom prompts guide the LLM to extract relevant movie information.

4. The AI generates a structured response based on the user's request.
